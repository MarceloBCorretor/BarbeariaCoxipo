# Barbearia Coxipó - Landing Page Interativa

Landing page premium e responsiva para a Barbearia Coxipó com sistema de agendamento conversacional via chat.

## 🎯 Características

- **Hero Assimétrico**: Design moderno com fundo em imagem profissional de barber
- **Chat Conversacional**: Assistente de agendamento interativo com fluxo natural de conversa
- **Tema Premium**: Paleta de cores sofisticada (Preto + Dourado Metálico)
- **Tipografia Elegante**: Oswald (títulos) + Poppins (corpo)
- **Animações Suaves**: Transições fluidas e efeitos de hover refinados
- **Responsivo**: Design mobile-first, funciona em todos os dispositivos
- **Seções Completas**: About, Serviços, CTA e Footer

## 🚀 Como Usar

### Instalação

```bash
pnpm install
```

### Desenvolvimento

```bash
pnpm dev
```

Acesse `http://localhost:3000` no seu navegador.

### Build para Produção

```bash
pnpm build
```

## 🎨 Design System

### Cores
- **Primária**: #D4AF37 (Dourado Metálico)
- **Fundo**: #0A0A0A (Preto Profundo)
- **Texto**: #FFFFFF (Branco Puro)

### Tipografia
- **Display**: Oswald Bold (700) - Títulos principais
- **Body**: Poppins Regular (400) - Texto corrido

## 💬 Chat de Agendamento

O assistente conversacional guia o cliente através de um fluxo natural:

1. **Saudação**: Boas-vindas e apresentação
2. **Seleção de Serviço**: Cliente escolhe entre Corte, Barba, Combo ou Hidratação
3. **Data**: Cliente informa a data desejada
4. **Horário**: Cliente escolhe um horário disponível
5. **Confirmação**: Revisão do agendamento
6. **Conclusão**: Confirmação com instruções
